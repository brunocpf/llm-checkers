# llm-checkers

An experiment with a Checkers game implemented in Next.js, exploring the use of large language models (LLMs) for AI decision-making and the Vercel AI SDK for seamless integration.

This project is using the new CSS anchor positioning API to create a responsive and interactive game board with animated piece movements.

## Technologies Used

- Next.js
- TypeScript
- Tailwind CSS
- Vercel AI SDK (model agnostic)
